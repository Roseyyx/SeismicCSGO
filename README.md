# SeismicCSGO

This repository contains the source code for an old cheat project, primarily intended for learning and experimentation purposes. It includes basic features such as:

- Aimbot
- Autowall
- Interface Manager
- Netvar Manager
- SDK
- Hooking with MinHook library
- Console loggin

**Please note:**  
- This is **not** designed for build or release.  
- The code is outdated and not maintained for production use.  
- Use it responsibly and ethically, primarily as a learning resource.
